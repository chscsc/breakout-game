# Breakout Mini game
![Screenshot of the game](screenshot.png)

### Usage
```shell
pip3 install pygame
python3 main.py
```

*or with virtual environments:*

```shell
python3 -m venv .venv
source .venv/bin/activate
pip3 install pygame
python3 main.py
```
